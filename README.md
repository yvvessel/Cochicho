# Cochicho 💬

Aplicativo mobile de mensagens em grupo desenvolvido com React Native, Expo e TypeScript.

O projeto foi criado com foco em aprendizado prático de desenvolvimento mobile Android, simulando funcionalidades presentes em aplicativos reais de conversa, como WhatsApp e Telegram.

---

# 📱 Funcionalidades

- Envio de mensagens em tempo real local
- Interface de conversa estilo WhatsApp
- Balões de mensagens personalizados
- Scroll automático com FlatList
- Tela de informações do grupo
- Alteração do nome do grupo
- Alteração da foto do grupo
- Navegação entre telas
- Persistência local com AsyncStorage
- Gerenciamento global de estado com Context API

---

# 🚀 Tecnologias utilizadas

- React Native
- Expo
- TypeScript
- Expo Router
- Context API
- AsyncStorage
- Expo Image Picker
- FlatList

O aplicativo foi desenvolvido com fins acadêmicos para a disciplina de Programação para Dispositivos Móveis Android.

# ⚙️ Como executar o projeto

## 1. Clone o repositório

```bash
git clone https://github.com/SEUUSUARIO/Cochicho.git
```

---

## 2. Entre na pasta

```bash
cd Cochicho
```

---

## 3. Instale as dependências

```bash
npm install
```

---

## 4. Inicie o Expo

```bash
npx expo start
```

---

# 📱 Executando no celular

Instale o aplicativo Expo Go:

https://expo.dev/go

Depois:

- escaneie o QR Code
- abra o projeto no celular
